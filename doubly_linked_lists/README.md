# Doubly linked lists

## Description

Implementation of a doubly linked list data structure with functions to add, traverse, sum, access, insert, and delete nodes.

## Key concepts

- Doubly linked list node structure (`prev`, `n`, `next`)
- Adding nodes at the beginning and at the end of a list
- Traversal, length, sum, indexed access, insertion, and deletion
- Proper memory management and freeing all nodes
