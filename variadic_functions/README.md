# Variadic functions

## Description

Exercises that demonstrate how to define and use functions that accept a variable number of arguments.

## Key concepts

- Using `stdarg.h` macros: `va_list`, `va_start`, `va_arg`, `va_end`
- Implementing a function that sums all its arguments
- Printing numbers and strings from variadic parameter lists
- Designing flexible function interfaces with variadic parameters
